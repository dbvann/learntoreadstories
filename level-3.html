Level 3: Blends & Digraphs

Now letters start teaming up. Some teams make one new sound, like sh. Some teams keep both sounds and slide together, like st and tr.

Pick one sound team, read one tiny story, and celebrate the win.
🐚

Digraphsh Stories

Read letter-team words like ship, shell, fish, and wish.

Start sh
🍟

Digraphch Stories

Practice words like chip, chop, chat, and lunch.

Start ch
🛁

Digraphth Stories

Read common th words like this, that, path, and bath.

Start th
🐋

Digraphwh Stories

Practice question words and patterns like what, when, and whale.

Start wh
🟦

Blendbl Stories

Read blend words like blue, black, blink, and blanket.

Start bl
👏

Blendcl Stories

Practice words like clam, cloud, cloth, and claws.

Start cl
⭐

Blendst Stories

Read words like star, stop, step, stick, and stone.

Start st
🚂

Blendtr Stories

Practice words like tree, trip, trail, train, and truck.

Start tr
Parent Tip: Digraphs make one new sound. Blends keep both sounds, but slide them together. Keep the practice short and let your child reread favorites.
