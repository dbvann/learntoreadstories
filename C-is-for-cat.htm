Level 1 — First Words

C is for Cat

Say one sound. Read a few words. Read one tiny story with Cat.

1. Say the Sound2. Read the Words3. Read the Story4. Celebrate
Step 1 · Say the Sound
C

C can say /k/, like cat.

cat
cup
cap
Step 2 · Read a Few Words
catc-a-t
cupc-u-p
capc-a-p
crabc-r-a-b
cakec-a-k-e
cavec-a-v-e
Step 3 · Tiny Story

Cat had a cup.

Cat had a cap.

A crab came by.

Crab saw a cake.

The cake was in a cave.

Crab can see it.

Step 4 · Celebrate
⭐

Now you learned C.

That’s a win.

If it still feels fun, read the story again.
